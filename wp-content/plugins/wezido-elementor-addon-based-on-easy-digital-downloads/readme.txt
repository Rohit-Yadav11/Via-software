 An Elementor Extension Works with the Easy Digital Download Products

== Description ==

Add more power to your Elementor page builder experience with EDD by using our easy to use widgets.
Consolidate these creative elements in your WordPress website to make it look more beautiful and friendly than before.

== Wezido - Elementor Addon Features ==

We have built 5+ useful widgets to enhance your Elementor Page Building experience. More coming soon!

1. [Headings](https://bit.ly/37cGOZm) - You will be able to choose any kind of heading tags.
also, there are many styles with a bunch of customization controls.

2. [Team](https://bit.ly/3tZHydU) - Wezido has team elements with customizable huge options.

3. [Flip Box](https://bit.ly/3rX2Ze5) - Flipbox available with bunch of customization control.
also flipbox has various animation options too.

4. [Before After](https://bit.ly/3aiQP9p) -There Available before after slider.
so you can easily show the slides.

5. [Blog Post](https://bit.ly/37fM7XQ) - You will be able to show your blog post based on category, tags etc.

6. [EDD Product](https://bit.ly/37daTbc) - You will be able to display your EDD digital products to any pages.


### Fully Customizable
Each widget comes with multiple options and controls, using which you can easily customize your designs as you want.

### Light Weight and Instant Loading
Code is optimized for fast loading and instant live editing with no extra resources.

### [Elementor](https://wordpress.org/plugins/elementor/) Page Builder is Required For This Plugin



== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= Where can i find the new element added =

New elements are added at the Wezido elementor widget area.

= Does it work with any theme? =

Absolutely! It will work with any theme where Elementor works.
